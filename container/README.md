# isse container


## Description

a docker image generation for isse, plus a script to run it.
it includes:
- [x] qp
- [x] cmake
- [x] gcc
- [x] qemu
- [x] openocd
- [x] st-flash and tools
- [x] arm-none-eabi-gcc and tools

## Getting Started

### Prerequisites

- [ ] docker
- [ ] docker-compose plugin
- [ ] add this to ~/.docker/config.json to avoid ctrl-p being captured by docker:
```json
{
  "detachKeys": "ctrl-@"
}
```
